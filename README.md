# 培棠.skill

> beta.skill——把一個人打包成一個檔案的嘗試。

🎞 **投影片(reveal.js)**:<https://tan9.github.io/beta.skill/>
(演講者模式按 `S`;中段有一張現場 `/beta` 實演的位置。)

這是我在現職單位倒數第二場分享留下的「實體成果」：一個真的跑得起來的 [Claude Code](https://claude.com/claude-code) skill，把我這快 19 年怎麼想事情、怎麼跟 AI 工作的方式，打包進一個 `SKILL.md`。

靈感來自 [colleague-skill](https://github.com/titanwings/colleague-skill)（同事.skill）。差別在於——那個是「被打包」，覺得自己要被取代；這個是我自己動手打包，而且我老早就想這樣。因為「可被取代」從來不是工程師的末日，是工程師的修養。

## 這是什麼

一個把「能寫成規則的我」交出去的檔案：

- **Part A——怎麼做事**：先問底層 primitive、宣告式優先、標準優先、怎麼跟 AI 吵架。
- **Part B——怎麼回話**：直接、乾、誠實到有點白目、不藏私。
- **Part Z——打包不進去的東西**：判斷該不該做、扛責任、為什麼還在乎——這三樣，留到最後一場親口講。

最後一場在這裡：[standardized-life](https://github.com/tan9/standardized-life)。

## 怎麼安裝

把它 clone 進你的 Claude Code skills 目錄：

```bash
git clone https://github.com/tan9/beta.skill.git ~/.claude/skills/beta
```

然後在 Claude Code 裡：

```
/beta
```

它不給你抄答案。它會先問你「這東西的底層是什麼」，再逼你從那裡推回來。
